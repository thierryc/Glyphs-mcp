Using the MCP ObjectWrapper Tools

This quickstart explains how to expose Glyphs SDK functionality via the MCP server plug-in.

1. Install the plug-in bundle

  1. Copy Glyphs MCP.glyphsPlugin into ~/Library/Application Support/Glyphs 3/Plugins/.
  2. Restart Glyphs to load the extension.

2. Start the MCP server

  1. Open the Edit menu.
  2. Choose Start Glyphs MCP Server.
  3. Watch the Macro Panel for the startup log, including the port allocation and the list of registered tools.

3. Connect an LLM client

  1. Point your MCP-compatible client (Claude Desktop, Continue, or other tooling) to http://127.0.0.1:9680/mcp/.
  2. Request the tool list to confirm the connection.
  3. Resources named resource://glyphs-mcp/docs/<page> expose this offline documentation.

4. Explore ObjectWrapper helpers

The ObjectWrapper exposes convenience helpers around Glyphs font objects. Common tasks include:

  • Iterating glyph masters and layers.
  • Reading and writing kerning groups.
  • Injecting anchors, components, and new outline paths.

5. Automate with Python snippets

Use the execute_code and execute_code_with_context tools to prototype automation scripts. Each execution runs inside Glyphs with the ObjectWrapper classes loaded so that they mirror the behaviour documented here.
