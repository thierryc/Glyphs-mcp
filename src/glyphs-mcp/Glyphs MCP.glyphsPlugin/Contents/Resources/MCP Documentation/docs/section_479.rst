.. attribute:: transform
		Transformation matrix.

		:type: :class:`NSAffineTransformStruct`

		.. code-block:: python
			# change transformation
			layer.backgroundImage.transform = ((
			    1.0, # x scale factor
			    0.0, # x skew factor
			    0.0, # y skew factor
			    1.0, # y scale factor
			    0.0, # x position
			    0.0  # y position
			))
