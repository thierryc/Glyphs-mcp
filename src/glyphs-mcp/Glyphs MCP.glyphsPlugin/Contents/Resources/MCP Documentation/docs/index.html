Glyphs SDK ObjectWrapper Overview

This documentation bundle packages the most relevant reference material from the Glyphs SDK ObjectWrapper for use inside the Glyphs MCP plug-in. The pages are static so that LLM operators can browse API details even when an internet connection is unavailable.

How to use these docs

The MCP server exposes this documentation bundle as structured resources. Agents can request resource://glyphs-mcp/docs/index.html for this overview, or query the JSON index at resource://glyphs-mcp/docs/index.json for a machine-readable manifest.

The bundle mirrors the layout generated by the Sphinx documentation build pipeline. Each page is lightweight HTML with headings and code examples designed to be LLM-friendly.

About the ObjectWrapper

The ObjectWrapper is the layer that translates native Glyphs App objects into Python-friendly wrappers. It provides convenience APIs for iterating glyph layers, manipulating paths, and accessing font metadata. The MCP server builds on these helpers to offer high-level tools such as kerning inspection, glyph mutation, and anchor management.
