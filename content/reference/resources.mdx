Resources are helper material for agents. They are not the main workflow surface; tools are.

## Registered resources

- Guide: `glyphs://glyphs-mcp/guide`
- Docs directory listing: `glyphs://glyphs-mcp/docs`
- Docs index: `glyphs://glyphs-mcp/docs/index.json`
- Kerning datasets listing: `glyphs://glyphs-mcp/kerning`
- Andre Fuchs relevant pairs snapshot: `glyphs://glyphs-mcp/kerning/andre-fuchs/relevant_pairs.v1.json`

## Bundled docs access

Use:

- `docs_search` to find relevant bundled Glyphs docs.
- `docs_get` to fetch only the page you need.

By default, individual docs pages are not registered as separate resources because that can flood `resources/list` in some clients.

Only call `docs_enable_page_resources` when you explicitly need per-page resource registration.
