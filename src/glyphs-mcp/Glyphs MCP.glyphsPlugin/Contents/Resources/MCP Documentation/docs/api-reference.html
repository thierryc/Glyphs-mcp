ObjectWrapper API Reference

The ObjectWrapper defines Python classes mirroring the native Objective-C objects inside Glyphs. Below are the classes most relevant to the MCP server.

Core classes

Class Description Key attributes
GSFont Represents an open font file, exposing masters, glyphs, and metadata. familyName, masters, glyphs, kerning
GSGlyph Stores glyph-level information, including unicode, category, and per-layer geometry. name, layers, leftKerningGroup, rightKerningGroup
GSLayer Contains outlines, components, and anchors for a specific master or special layer. paths, components, anchors, width

Helper methods

  • font.glyphs["A"].layers[master_id] – Access a glyph layer by master ID.
  • layer.paths[0].nodes – Iterate through the nodes within an outline path.
  • font.kerning[master_id][left_key][right_key] – Read a kerning pair value.
